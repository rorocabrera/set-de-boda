# Set de Boda - Music Set Constructor

A beautiful, interactive music set viewer and creator for weddings and events.

## Features
- **Set Creator**: Easily build music sets with songs and lyrics.
- **Interactive Viewer**: Click songs to view big, scrollable lyrics.
- **Navigation**: Smooth transitions and easy navigation between songs.
- **Customizable**: Color coding for song cards.
- **Offline Ready**: Saves sets to your browser's local storage.

## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the link shown in the terminal (usually http://localhost:5173).

## Deployment
To deploy this as a static website (e.g., on Vercel, Netlify, or GitHub Pages):
1. Build the project:
   ```bash
   npm run build
   ```
2. Upload the `dist` folder to your hosting provider.
